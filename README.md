# Sophie — Propuesta

Archivos en este proyecto:

- `index.html` — página principal con el mensaje, botones y modal.
- `style.css` — estilos en tonos celestes y animaciones.
- `script.js` — lógica para el botón de rechazo (se mueve) y el modal de aceptación.

Cómo abrir localmente (con XAMPP):

1. Asegúrate de que Apache esté corriendo en XAMPP.
2. Abre en el navegador:

```
http://localhost/sophie/index.html
```

Notas:
- El botón "No, gracias" se mueve aleatoriamente cuando intentas tocarlo o pasar el cursor.
- El botón "Sí, quiero" abre un modal con un anillo y un GIF romántico.
- La página está optimizada para móvil con botones grandes y diseño responsivo.

Si quieres que renombre archivos, añada confeti, o reemplace el GIF por uno tuyo, dímelo.
